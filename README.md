# object-linker

A CRUD interface for objects of different types which can be linked together by creating new relationship instances and associating it with the two objects along with arbitrary JSON data that can be optionally attached to the relation. Additionally a user can interactively create new instances of Content Type(s), Content Instance(s), and Content Attribute(s) through the interface.
